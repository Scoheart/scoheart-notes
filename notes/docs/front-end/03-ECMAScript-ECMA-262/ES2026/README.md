# ECMAScript 2026 (ES17)

> 状态: 提案阶段

## 概述

ECMAScript 2026 的特性目前处于早期提案阶段,具体内容尚未确定。

## TC39 提案流程

ECMAScript 的新特性需要经历以下阶段:

- **Stage 0** - Strawperson (初步想法)
- **Stage 1** - Proposal (提案)
- **Stage 2** - Draft (草案)
- **Stage 3** - Candidate (候选)
- **Stage 4** - Finished (完成,将在下一版本发布)
