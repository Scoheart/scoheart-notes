# 408 计算机学科基础综合课程大纲

## 课程概述
408计算机学科基础综合考试是计算机专业研究生入学考试的专业课科目，包含四门核心课程：
1. 数据结构
2. 计算机网络
3. 操作系统
4. 计算机组成原理

## 详细目录结构
