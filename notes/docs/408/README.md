# 计算机基础 408 目录

## 目录结构

```
├── data-structures-and-algorithms (数据结构与算法)
│   ├── fe (前端算法)
│   ├── leetcode (LeetCode题解)
│   └── leetcode_cn (LeetCode中文题解)
├── computer-networks (计算机网络)
│   ├── cache.md (缓存技术)
│   ├── cors.md (跨域资源共享)
│   └── http (HTTP协议相关)
├── operating-systems (操作系统)
│   ├── disk.md (磁盘管理)
│   ├── file.md (文件系统)
│   ├── index.md (操作系统概览)
│   ├── image.png (操作系统示意图)
│   └── shell.md (Shell相关)
└── computer-organization (计算机组织)
    └── index.md (计算机组织相关内容)
```

## 说明

本目录结构根据 alibaba/408.md 中指定的408计算机学科基础综合考试四大科目进行分类:
1. Data Structures and Algorithms (数据结构与算法)
2. Computer Networks (计算机网络)
3. Operating Systems (操作系统)
4. Computer Organization (计算机组织)

包含了数据结构与算法、计算机网络、操作系统和计算机组织等核心内容。
