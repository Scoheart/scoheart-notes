# ECMAScript 2025 (ES16)

> 状态: 提案阶段

## 概述

ECMAScript 2025 的特性目前还在提案和标准化过程中,尚未最终确定。

## 可能包含的特性

以下特性正在 TC39 提案流程中,可能会被纳入 ES2025:

### Stage 3+ 提案

1. **Temporal API** - 新的日期和时间 API
2. **Decorators** - 装饰器语法
3. **Record & Tuple** - 不可变数据结构
4. **Pattern Matching** - 模式匹配
5. **Pipeline Operator** - 管道运算符
