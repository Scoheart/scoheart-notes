# Exponentiation Operator (指数运算符)

> ECMAScript 2016 (ES7)

## 概述

ES2016 引入了指数运算符 `**`,用于计算幂次方。

## 基本语法

```javascript
base ** exponent
```

## 示例

```javascript
console.log(2 ** 3);    // 8
console.log(3 ** 2);    // 9
console.log(2 ** 10);   // 1024

// 等价于
console.log(Math.pow(2, 3));  // 8
```
