# Symbol.prototype.description

> ECMAScript 2019 (ES10)

直接访问 Symbol 的描述。

```javascript
const sym = Symbol('my symbol');

console.log(sym.description);  // 'my symbol'
console.log(sym.toString());   // 'Symbol(my symbol)'
```
